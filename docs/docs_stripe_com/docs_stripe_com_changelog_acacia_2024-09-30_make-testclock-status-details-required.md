# Makes status details for Test Clock test helpers required

## What’s new

Adds the required `status_details` hash to test clock resources to record
status-related information.

## Impact

Improves the accuracy of [Test Clocks](https://docs.stripe.com/api/test_clocks)
by requiring the `status_details` field. This ensures that status-related
information is available, allowing you to track and debug test clock activities
effectively.

toTestHelpers.TestClock.status_detailsChanged`optional → required`

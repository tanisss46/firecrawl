# Accounts now only show country-specific subfields for the legal_entity fieldBreaking changes

## What’s new

Updates the returned `Account` object to only show sub-properties of
`legal_entity` that are applicable to the account’s country, or that were
previously provided.

## Impact

The `Account` object now only shows relevant sub-properties of `legal_entity`,
providing more focused account information.
